# wildberries-sdk

В этом репозитории находятся сгенерированные SDK-клиенты Wildberries для разных языков на основе OpenAPI спецификаций Wildberries.

В данный момент представлены следующие разделы:

- General: `specs/01-general.yaml`
- Products: `specs/02-products.yaml`
- Orders FBS: `specs/03-orders-fbs.yaml`
- Orders DBW: `specs/04-orders-dbw.yaml`
- Orders DBS: `specs/05-orders-dbs.yaml`
- In-store pickup: `specs/06-in-store-pickup.yaml`
- Orders FBW: `specs/07-orders-fbw.yaml`
- Promotion: `specs/08-promotion.yaml`
- Communications: `specs/09-communications.yaml`
- Tariffs: `specs/10-tariffs.yaml`
- Analytics: `specs/11-analytics.yaml`
- Reports: `specs/12-reports.yaml`
- Finances: `specs/13-finances.yaml`
- WBD: `specs/14-wbd.yaml`
